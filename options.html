Get Updated Definitions

Update definitions occasionally, in order to pull down any new elements/classes that are used with various commenting systems.

Try this first if you notice that some comments are not hidden (especially if they previously were).

Excluded URLs

Include a list of URLs as regex patterns, one per line, that should be allowed to show comments all the time.

  • ^http://www\.example\.com/some_path$ - exact match
  • ^https?://www\.example\.com/some_path$ - exact match, either http or https
  • ^https?://(.*[.])?youtube[.]com - exact match, but works for http or https
  • ^http:// - any url using http
  • http://*/foo* - any url using http, and path starts with "/foo"
  • https://*.google.com/foo*bar - url using https, on google.com host, path starts with /foo, ends with bar
  • http://google.com/* - any url using http, and host is "google.com"
  • *://mail.google.com/* - any url using http or https, and host is "mail.google.com"
  • <all_urls> - matches any url that uses a permitted scheme
