Excluded URLs

Include a list of URLs as regex patterns, one per line, that should be allowed to show comments all the time.

  • ^http://www\.example\.com/some_path$ - exact match
  • ^https?://www\.example\.com/some_path$ - exact match, either http or https
  • ^https?://(.*[.])?youtube[.]com$ - ignore subdomain (i.e. works for youtube.com or blog.youtube.com)
  • ^http:// - any url starting with http
  • .* - any url (effectively disables the extension)
