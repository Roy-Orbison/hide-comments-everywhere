Excluded URLs

Include a list of URLs, one per line, that should be allowed to show comments all the time. Enter the exact URL or a match pattern.

  • http://www.whatever.com/some_path - exact match
  • http://*/* - any url using http
  • http://*/foo* - any url using http, and path starts with "/foo"
  • https://*.google.com/foo*bar - url using https, on google.com host, path starts with /foo, ends with bar
  • http://google.com/* - any url using http, and host is "google.com"
  • *://mail.google.com/* - any url using http or https, and host is "mail.google.com"
  • <all_urls> - matches any url that uses a permitted scheme
