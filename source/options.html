If one's available, you'll see a banner on this page. To get it, press the button below.

(Updates are also automatically installed when a new window is opened, or when Chrome is restarted.)

Filters

You can specify a list of URLs to not block, which should be allowed to show comments all the time.

Each URL must be a regex pattern, one per line. Here are some examples:

exact match ^http://www\.example\.com/some_path$
exact match, either http or https ^https?://www\.example\.com/some_path$
ignore subdomain (i.e. works for youtube.com or blog.youtube.com) ^https?://(.*[.])?youtube[.]com$
any url starting with http://example.com ^http://example\.com
any url starting with http ^http://
any url (effectively disables the extension) .*

Support

Questions? Problems?

If you have a question or problem, just open a new issue and I'll address it as time permits (or create a pull request if you have a fix).

If you've been using this extension for awhile and you like it, it'd be great if you could take a moment to rate it in the store .

Got a Site to Block?

If you have a site you think should be added to the global list, open an issue for it with as many details as possible.

If you know which element(s) need to be hidden, feel free to just open a pull request .
